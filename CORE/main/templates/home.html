About CORE

CORE is a request and donation platform that connects people in need with available resources and services in their community. Whether you need food, clothing, or support services, CORE makes it easy to submit a request or find assistance nearby.

Our mission is to make essential resources accessible and efficient for those who need them most, fostering a supportive network within the local area.

How It Works

1. Submit or Browse Requests

People in need can submit requests for resources, while providers can browse requests and fulfill them.

2. Connect & Coordinate

Our secure messaging system lets you coordinate pick-ups, deliveries, and details seamlessly.

3. Make a Difference

By participating, you're contributing to a stronger, more supportive community.

Platform Features

User Profiles

Create and manage profiles for both requesters and providers, with customized options for each.

Request Matching

Our system automatically matches requests with available providers based on location and type of need.

Notifications

Get instant notifications when a request is accepted, a message is received, or a request is fulfilled.
